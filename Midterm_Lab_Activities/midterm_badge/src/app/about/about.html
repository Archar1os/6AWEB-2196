The Community Help Desk Portal is a small multi‑page Angular application built as part of the Midterm Badge project. It demonstrates routing, shared services, observables, and pipes while fetching live JSON data from a remote API.

  • ✔ Angular Router for navigation across Home, About, Services, and Contact pages
  • ✔ Shared service fetching JSONPlaceholder posts and exposing them as Observables
  • ✔ Built‑in pipes (uppercase, date, slice) and a custom truncate pipe
  • ✔ Navigation UI with active route highlighting
